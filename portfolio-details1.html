Sleep Apnea Detection using AI

  • Category: Deep Learning
  • Project date: March 2025
  • Code URL: GitHub Repository

Objective

This project aims to detect sleep apnea using AI-based models trained on polysomnographic data. The system analyzes breathing patterns and detects anomalies associated with sleep apnea using CNNs and LSTMs.


Tech Stack

Python, TensorFlow, Keras, NumPy, Pandas, OpenCV


Deep Learning Approach

The model leverages CNN-LSTM hybrid architecture to analyze breathing patterns over time, ensuring high precision in apnea detection.


CNN-LSTM Model for Sleep Apnea Detection


Results & Performance

The model achieved an accuracy of 92% in detecting apnea events, outperforming traditional approaches. Data augmentation and transfer learning techniques were applied to improve model robustness.


Future Work

The next steps involve real-time monitoring applications and integrating the model into IoT-based wearable devices for real-world deployment.
