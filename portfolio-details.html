Audio-Based Emotion Prediction

This project leverages deep learning techniques to classify emotions based on audio input. Utilizing a dataset of 7,442 samples, the model achieved an accuracy of 87%. Feature extraction was done using MFCCs, and the model was implemented using TensorFlow and Keras.
